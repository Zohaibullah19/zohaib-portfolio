# 🚀 Zohaib Ullah Portfolio

A modern and responsive developer portfolio built with React and Vite, showcasing my skills, projects, education, certifications, and professional experience.

* Responsive Design
* Dark / Light Theme Toggle
* Professional Loading Screen
* Smooth Navigation
* Project Showcase with Modal
* Resume Download
* Contact Section
* AOS Scroll Animations
* Mobile-Friendly Layout
